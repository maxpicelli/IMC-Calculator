# IMC Calculator – GitHub Page

Página de apresentação profissional do app **IMC Calculator** para macOS, voltada para nutricionistas e pacientes.

## 🎯 Objetivo
Página marketing/landing page estilo GitHub Pages para divulgar o app IMC Calculator e centralizar o download das releases no GitHub.

## ✅ Funcionalidades Implementadas

### Seções da página:
- **Hero Section** – Gradiente animado em azul mais claro, ícone do app, badge macOS, screenshot em mockup macOS, estatísticas e CTA para releases
- **Seção de Recursos** – 6 cards animados com hover: IMC, Controle Diário, Importar do iPhone, Relatórios, Multi-usuário, Exportar
- **Para Quem É** – Dois cards: Nutricionistas (azul) e Pacientes/Atletas (verde)
- **Tabela IMC** – Tabela interativa com classificações OMS e barras de progresso animadas
- **Screenshot** – Mockup macOS com screenshot real do app
- **Apoie o Projeto** – Cards de Buy Me a Coffee/PayPal e Pix com botão copiar
- **Footer** – Contato, copyright e tagline

### Features técnicas:
- ✅ Navbar fixa com blur/glassmorphism e menu mobile
- ✅ Gradiente animado no hero com partículas CSS
- ✅ ECG/pulsação cardíaca decorativa em SVG
- ✅ Scroll reveal com IntersectionObserver
- ✅ Barras de progresso da tabela IMC com animação ao entrar na tela
- ✅ Botão "Copiar Chave Pix" com feedback visual
- ✅ Botão scroll-to-top
- ✅ 100% responsivo (mobile-first)
- ✅ Arquivo único index.html
